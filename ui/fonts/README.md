# Fantasy RPG Fonts

This directory contains optional fantasy-themed fonts for the AvaSim UI.

## Recommended Fonts

Download these free fonts from [Google Fonts](https://fonts.google.com/) for the best fantasy RPG aesthetic:

### Headings and Titles
- **Cinzel** - Roman capitals, ornate and medieval
  - Download: https://fonts.google.com/specimen/Cinzel
  - Files needed: `Cinzel-Bold.ttf`, `Cinzel-Regular.ttf`

- **Crimson Text** - Medieval serif, elegant
  - Download: https://fonts.google.com/specimen/Crimson+Text
  - Files needed: `CrimsonText-Bold.ttf`, `CrimsonText-Regular.ttf`

### Body Text and UI Labels
- **Spectral** - Readable serif with character
  - Download: https://fonts.google.com/specimen/Spectral
  - Files needed: `Spectral-Regular.ttf`, `Spectral-SemiBold.ttf`

- **Marcellus** - Clean serif for UI elements
  - Download: https://fonts.google.com/specimen/Marcellus
  - Files needed: `Marcellus-Regular.ttf`

## Installation

1. Download the font files (.ttf) from Google Fonts
2. Place them in this `ui/fonts/` directory
3. The application will automatically load them at startup

## Fallback Fonts

If you don't install these fonts, the application will fall back to system fonts:
- **Serif fallbacks**: Georgia, Times New Roman, serif
- **Sans-serif fallbacks**: Arial, Helvetica, sans-serif

The fallbacks will still work but won't have the same medieval fantasy aesthetic.
